# 🎬 YouTube Clone using React.js

A YouTube homepage and video player clone built with **React.js** and 
**Material UI**, featuring a responsive video grid, live search, and an 
embedded video player.

🔗 **Live Demo:** [Add your Vercel link here]

## ✨ Features
- Responsive video grid layout (MUI Grid)
- Live search filtering by title/channel
- Dynamic video player page with React Router
- YouTube-style navbar and sidebar

## 🛠️ Tech Stack
- React.js
- Material UI (MUI)
- React Router DOM
- React Player

## 🚦 Getting Started
\`\`\`bash
git clone https://github.com/alphonsa33/youtube-clone-react.git
cd youtube-clone-react
npm install
npm start
\`\`\`
